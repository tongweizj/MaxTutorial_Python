# Python 数据结构

## 元组 tuple

### 元组的创建

``` python

a = (1, 2, 3)
c = 1，

b = [1, 2, 3],  # 这就创建一个tuple，而不是list

print(a, type(a))
print(b, type(b))

```

### 元组的访问

``` python
# 索引
print(a[1])
print(a[-1])

# 切片
print(a[1:])

```
